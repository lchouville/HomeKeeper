# HomeKeeper
**Une application web légère pour gérer ses finances personnelles et ses listes de courses**, conçue pour être simple, sécurisée et auto-hébergée.

## ℹ️ Introduction
HomeKeeper permet de :
- **Suivre ses dépenses et recettes** via un tableau de bord intuitif.
- **Créer et organiser des listes de courses** (par catégorie, par magasin, etc.).
- **Visualiser son budget** grâce à des graphiques clairs.
- **Sécuriser ses données** avec une authentification utilisateur et des règles d’accès strictes.
